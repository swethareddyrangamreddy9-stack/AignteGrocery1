# AignteGrocery1
  # Project 49: Grocery Delivery Fulfillment Bot

## Tech Stack
- Python
- Streamlit
- Google Gemini 2.5 Flash API

## How to Run
1. Install dependencies: pip install -r requirements.txt
2. Run the app: streamlit run pharma.py
3. Enter your Gemini API Key in the sidebar.

## Features
- Explains Picking, Packing, and Dispatch logic.
- Built-in FAQ for instant answers.
- Safe GenAI implementation (No order tracking/placement).
